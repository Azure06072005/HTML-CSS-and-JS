ID selector

ID selector: khi chỉ muốn áp dụng CSS cho một phần tử chỉ định nào đó của HTML -> dùng id
ID là duy nhất theo quy chuẩn không cho phép đặt trùng id đã có check code hợp lệ tại https://validator.w3.org/
ID, class không được đặt tên bắt đầu bằng số, nếu có 2 kí tự trở lên nên đặt dấu _ hoặc - để nối

Class selector

Class selector: Do không thể sử dụng lại tên id -> trường hợp muốn khai báo
