87.5 z-index Hướng dẫn từ A đến Z về cách sử dụng z-index trong CSS

Các giá trị của z-index có thể là số nguyên (bao gồm cả số âm, số 0, và số dương) hoặc các từ khóa auto và inherit.
Số nguyên: Phần tử với z-index lớn hơn sẽ được xếp chồng lên trên phần tử có z-index nhỏ hơn.
auto: Phần tử sẽ tuân theo thứ tự xếp chồng mặc định.
inherit: Phần tử sẽ kế thừa giá trị z-index từ phần tử cha.
