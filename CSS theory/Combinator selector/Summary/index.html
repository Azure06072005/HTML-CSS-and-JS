Combination Selectors in CSS

Selector Description Example Meaning
Descendant Selector (space) Selects all elements that are descendants of a specified element. div p Chọn tất cả thẻ là con của thẻ được khởi tạo
Child Selector (>) Selects all elements that are direct children of a specified element. ul > li Chọn tất cả thẻ là con của thẻ được khởi tạo trước đó
Adjacent Sibling Selector (+) Selects an element that is directly after another specified element. h1 + p Chọn thẻ đứng ngay sau
General Sibling Selector (~) Selects all elements that are siblings of a specified element. h1 ~ p Chọn tất cả thẻ đứng sau (không cần sát)
Group Selector (,) Applies the same styles to multiple elements. h1, h2, h3 Chọn tất cả các thẻ có gắn tên
